# Steam Games Parser

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Простий та ефективний парсер для отримання інформації про ігри з магазину Steam.

## Опис

Цей проект надає зручний інструмент для отримання актуальних даних про ігри з платформи Steam. На даний момент він дозволяє швидко отримувати інформацію, таку як:
- Поточна ціна та наявність знижок
- ID гри
- Видавці та розробники
- Іншу корисну інформацію

Ідеально підходить для аналізу ринку, відстеження знижок та персонального використання.

## Функціонал (CLI)

Поточна версія з командним рядком (CLI) підтримує:
*   Пошук ігор за назвою
*   Вибір регіону та валюти для відображення цін
*   Вивід основних даних у зручному форматі
*   Багаторазовий пошук без перезапуску програми

## Функціонал (GUI версія)

**Активно працюю над версією з графічним інтерфейсом на основі PyQt6.**

Заплановані покращення:

*   **Зручний візуальний інтерфейс** для комфортної взаємодії
*   **Покращена обробка помилок** та інтуїтивне керування
*   **Графічне відображення даних**

## Встановлення та Запуск

1.  Клонуйте репозиторій:
    ```bash
    git clone https://github.com/Dargram/steam_parser
    cd steam_parser
    ```

2.  Встановіть залежності (для CLI версії):
    ```bash
    pip3 install requests rich
    ```

3.  Запустіть CLI-версію:
    ```bash
    python CLI_parser.py
    ```

## Використання

1.  **Введіть назву гри** - будь-якою мовою
2.  **Оберіть країну** для відображення ціни у відповідній валюті (`ua`, `de`, `us`)
    
    ⚠️ **Увага:** Деякі ігри можуть бути недоступні в певних регіонах (наприклад, "Postal 2" заборонений у Німеччині)
3.  **Отримайте дані** - інформація виводиться у зручному форматі
4.  **Продовжуйте пошук** або введіть `exit` для завершення роботи

## Технології

- Python 3
- Requests (для HTTP-запитів)
- Rich (для кращого відображення в консолі)
- PyQt6 (для майбутньої GUI версії)
